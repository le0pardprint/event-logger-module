# event-logger-module
Модуль уведомлений и событий для организации

## Назначение
Централизованная система аудита изменений для информационных систем организации. Решает проблемы отсутствия прозрачности в системах учёта заявок, бронирования помещений и выдачи оборудования.

## Основные функции
- фиксация событий с уровнями важности: Info, Warning, Error
- поддержка различных типов бизнес-событий
- многоканальное сохранение (файл + консоль)
- фильтрация событий по периоду, пользователю, типу
- восстановление состояния при запуске приложения
- обработка ошибок и некорректных данных

## Технологии
- .NET 6.0
- C#
- xUnit (для тестирования)
